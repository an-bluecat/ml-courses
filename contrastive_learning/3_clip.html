Contrastive Learning — Chapter 3: CLIP

Images and text in the same embedding space

The Key Insight

SimCLR: image paired with augmented image

positive pair = (cat_photo_view1, cat_photo_view2) // Both are the same image. Labels come from augmentation.

CLIP: image paired with its caption

positive pair = (cat_photo, "a cat sitting on a mat") // Different modalities! Labels come from the internet. // 400 million (image, text) pairs scraped from the web. // No human annotation needed.

Two separate encoders, one shared embedding space

image_encoder = ViT or ResNet → 512-dim vector text_encoder = Transformer → 512-dim vector   // After training: // "a cat" text embedding ≈ cat photo embedding (close in space) // "a dog" text embedding is far from cat photo embedding   // Same InfoNCE loss as before, just across modalities

Zero-Shot Classification — The Killer App

Classical classification: must train a head per dataset

// Train ImageNet classifier → only works on ImageNet 1000 classes // Want to classify medical images? Satellite images? Retrain. // Every new task = new labeled data + new training run.

CLIP zero-shot: no retraining, no labeled examples

// You want to classify cat vs dog. No labeled examples.   // Step 1: Encode the image img_emb = image_encoder(cat_photo) // → 512-dim vector   // Step 2: Encode candidate class labels as text text_emb_cat = text_encoder("a photo of a cat") // → 512-dim text_emb_dog = text_encoder("a photo of a dog") // → 512-dim   // Step 3: Pick the most similar text sim(img_emb, text_emb_cat) = 0.91 sim(img_emb, text_emb_dog) = 0.12 prediction = "cat" // highest similarity wins   // Add any label you want — no retraining needed: text_encoder("a photo of a Maine Coon cat breed") // works! text_encoder("a satellite image of a flooded city") // works!

Why this is remarkable

// CLIP trained on internet images+captions (no labels) // Tested on ImageNet zero-shot (never seen ImageNet labels) // Result: ~76% accuracy // // ResNet-50 trained WITH ImageNet labels: ~76% accuracy // // Zero labeled examples vs. 1.28 million labeled examples // Same accuracy. That's the power of scale + contrastive learning.
